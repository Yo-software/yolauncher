
# 🛠 Инструкция по установке и запуску (Node.js)

Поскольку **YoLauncher** построен на базе Node.js, процесс сборки максимально прост и привычен для веб-разработчиков.

## 📋 Предварительные требования

Для работы тебе понадобятся:

1.  **[Node.js](https://nodejs.org/)** — рекомендуем версию **LTS** (например, 18 или 20).
2.  **Менеджер пакетов:** `npm` (идет в комплекте с Node) или `yarn`.
3.  **Git** — чтобы склонировать код.

---

## 🚀 Как запустить в режиме разработки

1.  **Клонируй репозиторий:**
    ```bash
    git clone https://github.com/Yo-software/yolauncher.git
    cd yolauncher
    ```

2.  **Установи зависимости:**
    ```bash
    npm install
    ```

3.  **Запусти лаунчер:**
    ```bash
    npm start
    ```
    *Если проект использует Electron, эта команда откроет окно приложения.*

---

## 📦 Сборка готового приложения (.exe)

Чтобы собрать лаунчер в один файл для пользователей:

```bash
npm run build
```
Или, если используется `electron-builder`:
```bash
npm run dist
```
Готовые файлы появятся в папке `dist` или `out`.

---

## ⚙️ Настройка под свой сервер

Все основные настройки (ссылки на скачивание Minecraft, скины, API) обычно находятся в файле `config.json` или `.env`. 

1. Скопируй пример конфига (если есть): `cp config.example.json config.json`.
2. Отредактируй его под свои нужды.

---

## 🆘 Возможные проблемы

* **Ошибка `node-gyp`:** Если при установке зависимостей вылетают ошибки, убедись, что в системе установлены инструменты сборки (для Windows это `npm install --global windows-build-tools` или установленная Visual Studio с C++ компонентами).
* **Версия Node:** Если что-то не работает, проверь версию (`node -v`). Проект может требовать версию не ниже 16.0.
